## 사용 순서 (30분이면 1차 테스트 가능)

### 1. 접속
- Tailscale 켠 뒤 관리자 페이지 로그인
- (주소는 본인에게 전달받은 `http://100.x.x.x:8787/admin/saenggibu`)

### 2. 샘플·패턴 (최초 1회)
1. **샘플·패턴** 탭
2. 예전에 쓴 **현사윤 / 윤사 엑셀**이 있으면 TSV로 올리거나, 이미 올려둔 샘플 확인
3. **분석 실행** (Gemini 정제는 선택)
4. 샘플 목록에 항목이 보이면 OK

### 3. 학생 등록
1. **학생** 탭 → **📥 학생 TSV 샘플** 또는 **📥 현윤·윤사 TSV** 다운로드
2. 엑셀에서 열어 **실제 학생 2~3명만** 먼저 채우기 (전원 X)
3. **TSV 업로드**

#### TSV 열 이름 (중요)

| 열 | 2학년 현윤 | 3학년 윤사 |
|----|------------|------------|
| 세특 | `세특_현대사회와윤리` | `세특_윤리와사상` |
| 공통 | `name` `grade` `class_num` `number` | 동일 |

**활동 메모**는 짧은 키워드 여러 개를 **`;`** 로 구분:

```
아리스토텔레스 인물 탐구; 이상사회 발표; 비주얼씽킹 사성제
```

### 4. 작성
1. **일괄 작성** 또는 학생 목록에서 **작성** (1명씩 추천)
2. **보기** → 생성된 JSON/문장 확인
3. NEIS에 넣기 **전** 아래 체크리스트로 검토

- [ ] **과목 특성** — 현윤은 현대 이슈, 윤사는 고전·성찰 느낌이 난다

- [ ] **바로 제출 가능** / **30% 수정** / **다시 작성 필요**
