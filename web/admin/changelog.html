  • API 강제 재빌드 — --full-build / NAS-업데이트-API.bat가 실제로 이미지를 다시 빌드하도록 수정
  • git sync 후 재실행 — curl로 받은 배포 스크립트 대신 repo 최신본으로 재실행
  • 8787 포트 충돌 — gateway/API 레이아웃 전환 시 남은 컨테이너 정리
  • 점검 페이지 — 배포 실패 시 maintenance.on이 남지 않도록 처리

v2.0.1

2026.07.06

AI 맞춤법 검사, 배포 점검 페이지, 반·번호 선택 UI

기능

  • AI 맞춤법 검사 — 행발·세특·창체 작성 직후 맞춤법·띄어쓰기 자동 교정
  • 미작성 일괄 작성 — 이미 작성된 세특은 건너뛰고, 브라우저 초안 반영
  • 스마트 배포 — UI만 변경 시 Docker 재빌드 생략, 터널 불필요 재시작 줄임

UI

  • 반·번호 선택 — 타일/드롭다운 방식 (1~15반, 1~30번, 직접 입력)
  • 작성 팝업 — 중복 가로 프로그레스바 제거, 단계 메시지 정리
  • 배포 중 점검 페이지 — 502 대신 「잠시 점검 중」 안내 표시

인프라

  • sgb-gateway — nginx 게이트웨이 + Cloudflare 터널 sgb-gateway:8787 구성
  • src 변경 감지 — Python 코드 변경 시 API 이미지 자동 재빌드

v2.0.0

2026.07.05

4단계 교사 흐름, 비동기 일괄 작성, 검사기·개인정보 기반

기능

  • 4단계 흐름 — ① 스타일 학습 → ② 스타일 설정 → ③ 학생 등록 → ④ 작성·검토
  • 비동기 일괄 작성 — 여러 학생·항목 작성 시 진행 팝업 + job 폴링
  • 생기부 검사기 — 세특 분량(1500byte)·확인 필요 항목 체크리스트
  • NEIS 붙여넣기 — 탭 구분 행·【항목】 블록을 메모에 반영
  • 봉사·진로 UI — 창체 항목 확장, 등록 후 메모 수정

보안 · 데이터

  • 학생 데이터 암호화 — 저장 시 AES 암호화, 로드 시 복호화
  • PII 마스킹 — 생성·로그에서 개인정보 마스킹 옵션
  • 로그인 유지 — 새로고침 후에도 관리자 세션 유지

버그 수정

  • 학생 추가 500 오류, F5 새로고침 Internal Server Error 수정
  • 세특 키·모델 표시 오류 수정
  • 샘플 목록 로딩·stale 토큰·patterns 읽기 오류 수정

정리

  • NEIS용 복사·미사용 export/import 등 dead code 제거
