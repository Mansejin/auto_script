v2.3.0

2026.07.08

NEIS 글자수 규칙 통일, 관리 화면 전면 정리

기능

  • NEIS 글자수 — neis-counter 규칙 적용 (한글 3byte, 엔터 2byte 등). 행발 900B · 세특·창체 1500B
  • 분량 표시 — 항목 헤더에 공백 제외 N자, 공백 포함 N자, N바이트
  • 모델 분리 — 일괄·개별 작성은 Flash, 샘플 문체·분량 분석은 Pro
  • 로컬 개발 — SGB_DEV=1 + scripts/dev-local.sh 실행 시 더미 학생(dev-demo, 김테스트) 자동 시드

UI

  • 헤더·푸터 — 상단은 브랜드·사용법·테마만. 개인정보·패치 노트·버그 신고·로그아웃은 하단으로
  • 스타일 학습 — 파일 선택·업로드·분석 시작 한 줄 툴바. 샘플 목록 페이지네이션(10/30/50)
  • 학생 등록 — 고급 입력(AI·TSV) 제거. 학년 세그먼트 선택 코너 깨짐 수정
  • 일괄 AI 작성 — 영역 세그먼트 컨트롤 + 전체 너비 CTA로 재디자인
  • 검토 목록 — 체크박스·학생명·상태·검사 배지·열기/삭제 버튼 세로 중앙 정렬
  • 상세 편집 — textarea 10줄, 에디터를 검사 패널 위로. 검사 문구 지적→검토, 결과 UI 컴팩트화
  • 패치 노트 — 푸터 링크·페이지 제목을 패치 노트로 통일

정리

  • 봉사 활동 UI·일괄 작성 항목에서 제거
  • 일괄 작성 인원 제한·검토 전체 선택·개발용 runLog 제거
  • 항목별 다시 쓰기·맞춤법·분량 맞추기·AI 수정 버튼 제거 (v2.2.0에서 추가됐던 항목 편집 액션)
