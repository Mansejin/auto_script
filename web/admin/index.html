    사용 방법
    
        
      
    
      
      
        
    1. ① 스타일 학습 — 예전 생기부 파일을 올리고 분석 시작
      2. 
        
    2. ② 스타일 설정 — 분석된 문체를 확인·수정
      3. 
        
    3. ③ 학생 등록 — 작성 항목 선택 → 이름·학년·반·번호 → 선택한 메모 입력 → 학생 추가
      4. 
        
    4. ④ 작성·검토 — 영역 하나씩 선택해 일괄 작성 → 검토·복사·엑셀 추출
      5. 
      
    
      
      
        
    • 학년은 1·2·3 버튼, 반·번호는 드롭다운에서 고릅니다.
      • 
        
    • 세특 메모는 과목명 필수, 진로·수행평가·주제·활동 내용은 비워 두면 없음 처리됩니다.
      • 
        
    • ③ 등록된 학생에서 선택 삭제·전체 초기화로 학생 데이터를 지울 수 있습니다.
      • 
        
    • ④ 검토 목록에서 작성본만 삭제(초기화)하거나 엑셀 추출할 수 있습니다.
      • 
        
    • 기본 설정에서는 작성본이 서버에 저장되지 않고 이 브라우저에만 보관됩니다. 탭을 닫기 전에 엑셀로 백업하세요.
      • 
        
    • 학생 메모는 서버에 암호화되어 저장됩니다. 자세한 내용은 상단 개인정보를 참고하세요.
      • 
        
    • 행발과 세특은 각각 따로 ④에서 실행하세요.
      • 
        
    • AI 메모 정리·TSV 일괄 업로드는 ③ 하단 고급 메뉴에 있습니다.
      • 
        
    • 작성 중에는 팝업이 뜨며, 완료될 때까지 기다려 주세요.
